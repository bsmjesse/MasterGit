Important Tips for Using the Sentinel-FM Software

Setting Your Computer Screen for Optimal Display

To permit optimal display of all Sentinel-FM features:

 

  1. From the Start menu, choose Settings | Control Panel | Display.

  2. In the Display Properties dialog, choose the Settings tab. Make sure that the screen resolution is set to 1024 by 768 pixels.

  3. Click Advanced.  On the General tab, from the Display  option, choose Normal size (96 DPI).

  4. Choose the appropriate compatibility option  and click OK.

Logging In and Logging Out

When you navigate to the Sentinel-FM website, the first screen you see is the login screen. Log in to Sentinel-FM using your user name and password. The Sentinel FM home page opens.

 

  • To work with the main map, from the links on the top of the screen, click Map; to view history information, click History, etc.,

  • To view this help file or the other two help files (Configuration Guide and User Guide), click Help.

  • To return to the home page, click Home.

  • To log out, click Logout.

 

Note: After having logged in if you do not use Sentinel-FM for a period of approximately 30 minutes, you will be automatically logged out, for security reasons.

Viewing Tips

Using the Back button of your browser interface is not recommended.

 

If you are having trouble viewing the bottom of a screen, press F11. The screen shifts upwards. To shift back down, press F11 again.
