The Map Screen

To display the Map screen from the home page or from anywhere else in the Sentinel-FM interface, click on the Map link on the top of any page. The Map screen opens.  When you first access it, it displays a map of North America. The diagrams below show the main features of the map screen (top map display, bottom vehicle grid and right alarm/message windows). The map displayed in the History screen has similar features. For more information on the Map screen, see topics under Working in the Map Screen in the User Guide.
