Glossary

The following table summarizes a list of words and definitions.  In most instances the User can click on the word and be linked to more information on the topic.

 

Word

Definition

alarm

notification in Sentinel FM that a sensor has been triggered, a geofence or a geozone has been broken, etc.

 

arm/disarm

commands that enable or disable the sensors in the box connected to a vehicle--these allow or disallow the security alarm system to take effect.

 

box, or vehicle box

the vehicle-mounted device, with GPS receiving module, microprocessor, and wireless modem that communicates over standard wireless communication network.  It is also connected to user-defined input sensors and outputs for monitoring and remote control.

 

breadcrumb trail

a display on the History map that visually links the different locations where a vehicle has been

 

command

preset actions that a dispatcher can send from Sentinel FM to the box that control the box itself or that get information from the box

 

fleet

a number of vehicles grouped together for organizational purposes

 

geofence

a circular area that you create for security purposes, for instance, if a vehicle is towed out of the geofence area, a "geofence broken" critical alarm is automatically sent to the dispatcher

 

geoZone

a rectangle area that you create for monitoring convenience, for instance, if a vehicle is entering or leaving the geoZone area, you can specify that a notification be sent to the dispatcher.

 

GPS

Global Positioning System, a satellite-based navigation system used for determining one's precise location and providing a highly accurate time reference almost anywhere on Earth

 

landmark

a location on the map that you create for quick reference for your company

 

message type

a name for any kind of communication between Sentinel FM and a vehicle box

 

output

a wire that Is connected to the box and is designed to allow a dispatcher to remotely control a vehicle event. For instance, a dispatcher can send an output to remotely lock the doors of the vehicle.

 

sensor

a wire that is connected to the box and designed to detect an event, for instance, whether the ignition of a car is on or off

 

vehicle grid

the table under the map display in the Map, History, or Messages screens
