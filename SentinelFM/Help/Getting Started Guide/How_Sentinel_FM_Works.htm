How Sentinel FM Works

The Sentinel-FM system utilizes:

 

  • GPS (Global Positioning System) technology that pinpoints the location of vehicles;

  • Wireless communication networks that permit communication between vehicles and the Sentinel-FM data servers;

  • Digital Mapping Systems that provide detailed street-level maps and street addresses;

  • An in-vehicle unit (the "blackbox") whose microprocessor manages various input sensors and output controls to allow a User to monitor and control the vehicle;

  • An optional MDT (Mobile Data Terminal) unit in the vehicle that allows 2-way text messaging; and,

  • The Internet to give users access to the Sentinel FM software at any time from anywhere, using a standard web browser.
     

           

 

The Sentinel-FM system can be used to:

 

  • Monitor a vehicle’s location, speed and direction, in real-time and historically;

  • Monitor customizable user-defined physical and virtual input sensors (door open/close, ignition on/off, etc.);

  • Control outputs (locking/unlocking doors, etc.);

  • Utilize two-way text messaging to improve fleet management operations;

  • Monitor and manage vehicle maintenance cycles;

  • Generate detailed reports (Trip Details, Trip Summary, History, Alarms, Stop and Idling, Messages, Off-hours Operation, Landmark Activity and Fleet Maintenance);

  • Create pre-defined landmarks and GeoZones (see Setting Up Landmarks and Setting Up GeoZones); and,

  • Set up instant event notification via pager, text messaging or email to support sensor monitoring such as ignition on, speed trigger, geofence break, landmark approach.  Notification can also occur when a vehicle is due for scheduled maintenance or requires special service (e.g. GPS antenna broken).
