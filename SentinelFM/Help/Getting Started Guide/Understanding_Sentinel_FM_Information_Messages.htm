Understanding Sentinel-FM Information Messages

The following table explains messages that you may see on the Sentinel-FM screen:

 

Wording of message

What it means

Communication with Vehicle(s): [Name of vehicle] is not available at this time. Displaying last known position.

The wireless network on the vehicle side may have no link or no coverage or bad signal quality at that time.

Communication with Vehicle(s): [Name of vehicle] is not available at this time and your message was not sent to them.

GPS Position for Vehicle(s): [Name of vehicle] is not valid. Displaying last known position.

GPS module in the box can not “see” enough GPS satellites to calculate the position, for instance, if the vehicle is in an area with many tall buildings, in an underground garage, or in a forest.

No Valid GPS Data

Address Resolution in Progress

The vehicle position coordinates are currently being resolved into an address or a landmark.

No GPS Data

No GPS data is provided in the specific message.

Position information for vehicle(s): name considered old according to parameter set in User Preferences

The GPS position for selected vehicle in the database is older then the number of days, hours and minutes set for "Consider position old after" setting in User Preferences.

Lat: [data], Lon: [data]

There is no street address available at the position. The system shows the latitude and longitude of the position.
