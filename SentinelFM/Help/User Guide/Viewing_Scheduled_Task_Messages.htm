Viewing Scheduled Task Messages

Scheduled tasks can be viewed by accessing the Sentinel FM Messages function and selecting the Scheduled Tasks option.

 

To view scheduled task messages in the Messages screen:

 

  1. From the home page, click Messages. The Messages screen appears.

  2. At the top of the screen, choose the Scheduled Tasks option.

    • In the From and To fields, specify a range of time for sent messages by choosing start and end dates and times. For dates, you can click in the calendar.

    • In the Fleet and Vehicle fields, choose whose messages you want to display. You can view messages for "All Vehicles" and for "Entire Fleet", or, to narrow your search, choose a fleet and a vehicle.

  1. Click View Scheduled Tasks. A grid appears displaying the scheduled task messages that you chose. In the grid, note the following columns:

    • In the Date/Time column, view the date and time the scheduled task was initiated.

    • In the Description column, the description of the in-vehicle unit is listed

    • In the Command column, the scheduled task is listed

    • In the Last Try Date column, the date/time of the last attempt to send the command is listed

    • In the Sent column, the date/time the scheduled task was completed

    • In the Status column, the current status of the scheduled task is listed (Pending, Completed, Expired)
