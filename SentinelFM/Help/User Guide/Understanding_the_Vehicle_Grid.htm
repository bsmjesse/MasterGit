Understanding the Vehicle Grid

The vehicle grid displays a list of the vehicles in a fleet. The grid appears on the Map screen when a fleet has been selected. For details on the Map Screen, see Displaying Vehicles in the Map Screen.

 

When you select several vehicles in a fleet, the vehicles you select are moved to the top of the list in the vehicle grid. To select or deselect all vehicles in the vehicle grid, enable or disable the checkbox at the top left of the vehicle grid.

 

To view the vehicle grid more easily, click Full Screen Grid. The vehicle grid is displayed in its own full screen window.  Alarms appear as moving text at the bottom of the screen. To return to the standard Map screen, from the full screen grid, click Close Window.

 

To control how often the vehicle grid information is updated from the database, configure the Screen Refresh Frequency option in User Preferences.  To update the vehicle grid information immediately, press F5.

 

Columns in the Vehicle Grid

The vehicle grid contains the following columns:
 

Name of Column

Function

Details

 

Click checkbox to select or de-select a vehicle.

Click header checkbox to select/de-select all vehicles

Commands / Info

Click on description of vehicle hyperlink to access commands and information for the selected vehicle.  

 

For details, see Viewing Sensors and Sending Outputs or Commands  Click on header title to sort the vehicle grid by vehicle description.

 

Address

Shows the latest address available from the Sentinel FM database.

This field can be configured to show street address or nearest landmark. See Changing User Preferences

Date / Time

Shows the last date and time the vehicle reported automatically, or the last time a User requested a vehicle position.  

 

Click on header  title to sort the vehicle grid by Date/Time.

Speed

Shows the speed and heading the vehicle was going at the last update.

 

 

Armed

Shows whether the box of a vehicle is armed or disarmed.

 

For details, see Viewing Sensors and Sending Outputs or Commands.

Status

Shows status of last vehicle position received.

 

Moving, Idling, Stopped, Parked, N/A.  Click on header title to sort the vehicle grid by Status.

 

Duration

Shows how long the vehicle has remained in its last status state.

 

 

 

 

Note that the information in the Address, Time, Speed, Heading, and Status columns is interactive, that is, when one element changes, the others may, too.
