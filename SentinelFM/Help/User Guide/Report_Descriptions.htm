Report Descriptions

Following is a list of reports available in Sentinel-FM. To generate a report, see Generating Reports.

Trip Details Report

Provides details about vehicle trips that took place in a specified period of time. (A "trip" is considered to begin when the ignition is turned on, and to end when it is turned off.) The report shows date, time, location, etc. for the trip start and the trip end, as well as vehicle position information that was communicated and sensors that were triggered during the trip. You can include or exclude street address, sensor triggers, scheduled position reports, stored position, idling time, and trip summary. You can generate the report for a single vehicle or for an entire fleet.

Trip Summary Report

Provides summary information about vehicle trips that took place in a period of time that you choose. (A "trip" is considered to begin when the ignition is turned on, and to end when it is turned off.) For each trip, shows departure address and time, arrival address and time, distance traveled, trip time, idling time and stop time. Provides totals for number of trips, trip time, idling time, stop time, and distance traveled in the selected period of time. You can include or exclude stored position. You can generate the report for a single vehicle or for an entire fleet.

Alarms Report

Provides details about all alarms that took place in a specified period of time. You can generate the report for a single vehicle or for an entire fleet.

History Report

Provides details about all activities that took place for a single vehicle that you choose over a period of time that you choose. Includes IP address updates, sensor triggers, commands, outputs, position updates, scheduled position reports, MDT text messages and Geo Fence violations. You can include or exclude scheduled GPS coordinates, sensor triggers, and invalid GPS positions.

Stop and Idling Report

Provides details about all stops and idlings that took place in a period of time that you choose. Includes arrival time, street address, departure time and stop and idle duration. Provides totals for number of stops/idlings and stop/idling time. You can filter by stop/idle  duration and you can include or exclude stored positions. You can choose to show stops only, idlings only or both.  You can generate the report for a single vehicle or for an entire fleet.

Messages Report

Provides a list of text messages sent and received by the system in a period of time that you choose. Includes From, To, direction of a message, message text and responses. You can generate the report for a single vehicle or for an entire fleet.

 

Off-Hours Operation Report

Provides a list of all vehicles that were used after hours over a period of time that you choose.  You can select what are considered off-hours periods during the work week and on the weekends to report on.  You can also choose to include/exclude coordinates, sensors and invalid GPS positions in the report.

 

Landmark Activity Report

Provides a summary of the total time spent by a vehicle at fleet associated landmarks, if any over a period of time that you choose.

 

Fleet Maintenance Report

Provides a report on vehicle maintenance including current odometer readings, and whether maintenance has been recently performed, is due or is overdue.
