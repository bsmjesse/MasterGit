Maintenance Complete

When scheduled maintenance has been performed on a vehicle, the User should select the Maintenance Complete hyperlink, and ensure that the service(s) displayed reflect the service(s)  that were performed.  Services can be added, modified or deleted using the drop-down list of pre-defined services or by simply entering, modifying or deleting any of the text in the display area below the drop-down list.  A maximum of 1024 characters can be entered.  

 

The User also has the option, before saving, of entering items requiring maintenance during  the next service cycle.

 

Once all information has been entered the User then clicks on the Save button to have that service information stored in the Sentinel-FM database.
