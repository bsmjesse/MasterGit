Displaying Vehicles on the Map Screen

To display vehicles on the Map screen, you must first have a fleet selected.  A fleet is a number of vehicles grouped together by a company for organizational purposes, such as "delivery trucks" or "personal cars". Grouping vehicles into fleets makes it easier to view them simultaneously on the map or to create a report about them.  Vehicles can be assigned to more than one fleet. A company can have several fleets, and every company can also have a fleet named "All Vehicles" that contains all the vehicles in the company (ability to view "All Vehicles" depends on your level of security assigned to User).

 

Fleets can be setup and configured by a customer. A customer can set up a new fleet, and move vehicles between fleets. To move vehicles between fleets, see Adding Vehicles to Fleets.

 

If you have not chosen a fleet before, the Map screen shows only a map. Once you choose a fleet, however, a vehicle grid appears below the map with detailed information about each vehicle in the fleet selected. If you primarily deal with one fleet only, you can configure that fleet as the default fleet in User Preferences so that it will be automatically pre-loaded when you enter the map screen.

 

To display vehicles on the map screen:

 

  1. From any Sentinel-FM page, click on the Map hyperlink on the top of the screen. The Map screen opens and displays the default fleet grid if configured to do so.

  2. If a default fleet has not been configured or you want to select an alternate fleet, from the Fleet list, choose the fleet whose vehicles you want to display. The vehicle grid displays a list of the vehicles in the fleet you choose.

  3. To locate a vehicle on the map, find it in the vehicle grid. In the left most vehicle grid column, enable the checkbox next to the vehicle to be located.   Vehicles in the vehicle grid can be sorted by Name, Date/Time or Status by clicking on the title hyperlink of the title in any of these columns in the vehicle grid.

  4. To view the last position of the vehicle stored in the Sentinel-FM database, click the Map It button.  The map zooms in on the vehicle position which is shown as an icon, and the vehicle you selected moves to the top of the vehicle grid. (When you choose this option, Sentinel-FM does not communicate in real-time with the box in the vehicle.)

  5. To request the vehicle's current GPS position with real-time data from the vehicle box, click the Update Position button. The map zooms in on the new vehicle position which is shown as an icon, and the vehicle you selected moves to the top of the vehicle grid. If no new data is available, the system notifies you and the map zooms in on the last vehicle position stored in the Sentinel-FM database.

 

Note: You can select several vehicles to locate, or select or deselect all vehicles in the fleet  by enabling or disabling the checkbox at the top left of the vehicle grid. The map will display the positions of all the vehicles you selected.

 

To have the vehicle position and vehicle grid information updated from the Sentinel-FM database on a regular basis,  configure the screen refresh frequency option in User Preferences.  To update the vehicle position and vehicle grid information immediately, press F5.

 

If a User cannot easily find a specific vehicle in a large vehicle grid, see Searching the Vehicle Grid.

 

If a User would like to view selected vehicles on a larger map the Full Screen Map button can be selected which opens a full screen map view.  When viewing vehicles in the full screen map view the information about the vehicle which is normally displayed in the vehicle grid can be displayed as a popup window by moving the mouse cursor over the vehicle icon.
