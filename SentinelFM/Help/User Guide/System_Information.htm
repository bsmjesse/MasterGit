System Information

On the right side of the Sentinel-FM home page there a hyperlink next to User: and a status column that displays current information about the Sentinel-FM system including, User name and organization, Last Login, System Updates and New Features and allows a User to modify their User Preferences.

 

 

 

The User Info section of the home page screen displays the name of the current logged in User, their company name and the last time they logged into the system.  A User Preferences hyperlink also exists that allows the logged in User to change their User Preferences and Password.

 

 

The System Updates section of the home page screen notifies the User of any expected system down times due to system upgrades or maintenance.

 

The New Features section of the home page screen informs the User of  any new features that are available in the current version of the system.
