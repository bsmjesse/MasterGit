Logging In

When a User launches their web browser and navigates to the Sentinel-FM website, the first screen they will see is the login screen.  Log in to the Sentinel-FM system using the user name and password that has been supplied to you by BSM.

 

Note:  To maintain client privacy and confidentially, client account information is protected by 128-bit encryption.  For best viewing of this website, please use Internet Explorer Version 5.0 or higher and a monitor resolution of 1024 x 768.  
ALL POP UP BLOCKING software must be disabled before starting the Sentinel-FM application.

 

Once a User has successfully logged into the system the Sentinel FM home page will be displayed which provides easy access to the various system functions through a series of horizontal of hyperlinks on the top of any Sentinel-FM page.

 

 

Takes the User back to the Sentinel-FM home page

Graphically review the historical movements of vehicles

Review maintenance history and specify maintenance service types and service intervals.

Review sent/received text messages and generate and send new messages

Review current Landmarks and GeoZones and generate new ones

Generate a series of standard reports

Configure the system including adding/modifying fleets, vehicles and users

Access to online help

Logs a User off the Sentinel-FM system

 

 

Note:  If you do not use the Sentinel-FM system for a period of approximately 30 minutes, you will be automatically logged out for security reasons.
