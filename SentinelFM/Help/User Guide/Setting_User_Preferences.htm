Setting User Preferences

From the User Preferences hyperlink on the Sentinel FM home page a User can set a variety of User-specific parameters including:

 

Setting

Explanation

Time Zone

In order to display local time in the system, you must select your time zone.

Daylight Savings Time

This box is checked when daylight savings time is in effect.

Km/Miles

Allows User to specify whether distances should be displayed in kilometers or miles.

Default Fleet

Allows User to specify default fleet that is automatically pre-selected whenever fleet information is accessed.

Show Landmark Name instead of address

Allows a landmark name instead of address to be shown on map display and in reports.

Show Read Messages

Determines whether messages continue to be displayed in the map screen messages window once they have been read.  By default they are shown there for 24 hours.

Alarm Refresh Frequency

Allows a User to specify the amount of time before new alarm information on the map screen alarm window is refreshed with new information from the system database.

Screen Refresh Frequency

Allows a User to specify the mount of time before the map display and vehicle grid are refreshed with new information from the system database.  Auto refresh must be enabled for this setting to take effect.

 

To change any of these User Preferences the User selects the Edit button, make the appropriate changes and t hen selects the Save button.
