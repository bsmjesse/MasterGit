Using WebGrid Functionality

The Sentinel FM system utilizes WebGrid.NET functionality by Intersoft Solutions to display and manage database information.  WebGrid has several built-in functionalities that can help a User display, filter and extract data in a more efficient and effective manner.

 

Column Sorting

 

Whenever data is being displayed using the WebGrid interface the data for any database column can be sorted in ascending, descending or no order by simply clicking on the column header. Each click will cycle the sort order through the three available states, ascending order, descending order and no order.  An arrow in the right side of the column will appear indicating that the column is sorted.  A down arrow indicates descending order and an up arrow ascending order and no arrow indicates the column is nor sorted.  Only one column can be sorted at a time.

 

Column Filtering

 

The WebGrid interface can be used to filter large datasets to find specific data a User is interested in.  Both single and multiple column filters can be defined.  When the WebGrid filtering option is available (it can be enabled or disabled for the main map screen in User Preferences) a row will appear below the column headers with blank fields and filter selection icons .

 

 

 

The User can type text to be filtered by in the blank field and then click on the filter selection icon to select the appropriate filter type (see left).  "Like" is the default filter type if one is not selected.  After choosing a filter type the User then selects "Apply All Filters" and the filter is applied and the results shown in the database grid.  Several columns can be filtered concurrently by repeating the process.  In the example above the Status column has been filtered for the text "Parked" and the Description column has been filtered for the text "Paul".   Only those database records matching both filters are displayed in the database grid..

 

The User can also clear a specific filter by selecting the filter type icon and choosing "Clear This Filter".  All filters can be cleared by choosing "Clear All Filters".

 

 

Exporting a Grid

 

Any data being displayed in a database grid using the WebGrid interface can be exported to a variety of file formats (see list below) by selecting the export grid button located at the bottom right of any WebGrid display .

 

The User selects the output file format and whether the output should be in Portrait or Landscape mode and then selects the "Export now".  

 

The export function will only export the database records that are currently being displayed on the first page of the database grid.  To export all records make sure that the "items per page" option is set to "All".

 

 

 

 

Navigating the Database Grid

 

When displaying large datasets through the WebGrid interface the data may be spread out over several pages depending on how many items per page are being displayed.  The User can move between the various database pages using the controls on the right bottom of the WebGrid display.
