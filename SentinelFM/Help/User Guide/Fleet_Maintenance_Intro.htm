Fleet Maintenance - Introduction

The fleet maintenance functionality in Sentinel-FM allows a User to specify and monitor odometer-based maintenance cycles for vehicles and be notified by email when a specific vehicle is due for maintenance.

 

The main fleet maintenance screen in Sentinel-FM allows a User to view information about the maintenance service that has occurred and that is planned for a specific vehicle.  The main maintenance screen shows the current odometer reading for each vehicle in your fleet,  the odometer reading for the last maintenance that was performed and the odometer reading for the next service scheduled to be performed.

 

 

On the main maintenance screen a User can also:

 

  1. View the detailed service history and next service details for a specific vehicle by selecting the View Details link.

  2. Specify when maintenance has been completed for a vehicle by selecting the Maintenance Complete link.

 

A User also has the option of generating fleet maintenance reports by selecting Fleet Maintenance Report from drop-down report list on the Reports screen.
