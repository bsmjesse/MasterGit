Messages - Introduction

The Sentinel FM messages function deals with three message types:  Text Messages, Alarms and Scheduled Tasks.

 

  1. Text messages refer to text sent between the Sentinel FM system and an in-vehicle unit equipped with a Mobile Data Terminal (MDT)

  2. Alarms refer to notifications received from a in-vehicle unit when an enabled sensor has been triggered

  3. Scheduled Tasks refer to tasks scheduled to be performed in the future such as a scheduled command or a scheduled text message sent to an in-vehicle unit that is currently not powered up.  

 

When the Sentinel FM Messages function is accessed from the hyperlink at the top of any page the Text Messages option is chosen by default and a list of messages sent/received over the last twenty-four hours is displayed for the system configured fleet.

 

A User has the option of working with Text Messages or choosing one of the other message options (Alarms or Scheduled Tasks) by clicking on the appropriate radial button.
