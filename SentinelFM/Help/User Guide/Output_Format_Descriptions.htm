Output Format Descriptions

Following is a list of output formats available in Sentinel-FM. To generate a report, see Generating Reports.

 

  • Crystal: A plain HTML format, not used very often, since it is not easy to print. Use Crystal if you don't have access to a PDF format.

  • PDF: The default output format.

  • Excel: Standard Microsoft spreadsheet format.

  • Word: Standard Microsoft word processing format.  

 

Note:  Adobe Acrobat Reader,  a free PDF viewer can be downloaded by selecting the "Get Adobe Reader" link located in the bottom right of the reports page.
