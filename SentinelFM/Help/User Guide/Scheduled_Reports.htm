Scheduled Reports

Reports can be scheduled to be run at times in the future either once, daily, weekly or monthly by selecting the "Schedule Report" button under the Reports menu.  The User can also the time the report is to be generated and whether it is to be emailed on stored to disk.  If the scheduled report is to be created on a recurring basis the User must specify the report life time  (when the Sentinel FM system should start generating reports and when it should stop.

 

 

Once a report has been scheduled a list of scheduled reports can be viewed by selecting the "View Scheduled Reports" button under the Reports menu.  A list of scheduled reports is displayed.  A User can delete a scheduled report by clicking on the delete icon

 

If the scheduled report is to be stored to disk rather then being emailed the User can select the hyperlink in the first column of the report list.  This will list stored reports and allow them to be downloaded to your local computer.
