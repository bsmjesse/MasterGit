Adding New Maintenance Plans

After defining Services and Notification Policies a User can create Maintenance Plans for vehicles.   Under the Maintenance tab, select the fleet and vehicle you want to create a maintenance plan for.  If the Odometer and Engine Hours are at zero you will need to initialize the current vehicle settings in order to use the maintenance function.  A User can click on the "Set" button in order to set the current vehicle engine hours value.
 

Note:     A User can change the vehicle odometer value by sending the "Set Odometer" command to the Sentinel FM in-vehicle unit.
 

 

To add a new maintenance plan:
 

  1. Click on the Add New Plan hyperlink

  2. Type in a plan description (mandatory)

  3. Select a Service Type (mandatory)

  4. Select a Defined Service (mandatory)

  5. Select a Defined Notification (mandatory)

  6. Select a Schedule Type (mandatory) -  If once is selected specify due value,  if recurring is selected specify due value, interval and end service value (optional)

  7. Type in email address or comma-separated email addresses to get notifications (mandatory)

  8. Type in any useful information to be added as notes  (optional)

  9. Click on the Save button to save the new plan

 

 

Once a new maintenance plan is entered it will be added to the due services list where it can be viewed, edited, deleted or closed by clicking on the "Get Due Services" hyperlink.  If a maintenance plan is created with a schedule type of once, when the odometer or engine hour "due value" has been reached the maintenance plan will be automatically moved into the Past Services list which can be viewed by clicking on the "Get Past Services" hyperlink.

 

If a maintenance plan is created with the schedule type as recurring, after the current odometer or engine hours passes the "due value" the appropriate maintenance plan(s) will be moved into Past Services which can be viewed by clicking on the "Get Past Services" hyperlink.  Since the schedule type is recurring, however, after the User closes this service then based on the set interval the next due value will be calculated and the service will automatically appear again under Due Services which can be viewed by clicking on the "Get Due Services" hyperlink.

 

Note:     If the schedule type is recurring it is critical that the User close the service once it passes the current value for the next due value to be calculated.
