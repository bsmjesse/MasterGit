View Details

When a User selects the View Details hyperlink on the Fleet Maintenance page they can review the service history of a particular vehicle and review and/or specify the next service required for the vehicle and who should be contacted by email when service is due.

 

The Service History table of the left side of the page displays all service that has taken place including Date/Time of service, description of service and odometer reading at time of service.  If the service history record is extensive the User will see numbers indicating the number of pages of the service history and can view alternatively pages by selecting the appropriate number.

 

 

The Service Info section of the screen allows a User to see what service(s) are scheduled as well as when those service(s) are scheduled to take place and the email of the person who should be notified when service is due.

 

 

A User can add a pre-defined scheduled service by selecting from the drop-down list and pressing the Add button.  The selected item is added as text into the display area below the drop-down list.  A User can also simply type in text about the service(s) to be performed in the display area below the drop-down list.  A User can also delete any of the text by selecting it in the display below the drop-down list and pressing the Delete button on the keyboard.  The maximum number of characters that can be entered in the display area is 1024.

 

The User can also specify the server interval (distance from the last service odometer reading) at which the next service(s) should take place.

 

If necessary, the User should also edit the Last Serviced Odometer reading to make sure it corresponds with the odometer reading of the last (most recent service) in the Service History list.

 

The User can also specify or modify the email address of the person who should be notified when service is due on a vehicle.  See Maintenance Email Notification for more details.
