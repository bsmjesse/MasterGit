Understanding Message Types

In the History screen, when you choose the Vehicle Path option for the history type, you can view locations for and information about messages, commands, GPS positions, triggered sensors, and other activity history that was communicated, either from the Sentinel FM server to the box in a vehicle, or vice versa. (For details about the Vehicle Path option, see Viewing Vehicle Path History.) All of these communications are referred to in the grid as "message types".

 

  • Data that was sent from Sentinel FM to a vehicle box is called "outgoing", and appears with a >> in the Message Type column.

  • Data that was sent from a vehicle box to Sentinel FM is called "incoming", and appears with a << in the Message Type column.
