Searching the Vehicle Grid

The vehicle grid displays a list of the vehicles in a fleet. The first fifteen vehicles in the grid are visible through scrolling on the first page of the  Map screen when a fleet has been selected. For details, see Displaying Vehicles in the Map Screen.

 

To search for a vehicle in a vehicle grid with a large number of vehicles, first check the Fleet list to be sure that you are looking in the correct fleet. If you are not sure which fleet to choose, choose "All Vehicles". Then do the following:

 

  1. In the Map screen, click Search.

  2. In the Vehicle Search screen that appears, choose whether you want to search by Vehicle Description, License Plate, Landmark, Box ID or Street Address.  If you choose to search by Landmark or Street Address, other fields appear where you can enter information to help your search, such as, for example,  the fleet and the distance from a specified landmark.

  3. Click Search Vehicles. The vehicle(s) that match your search are listed below the search fields.

  4. From the list, select the vehicle(s) you want to view by checking the appropriate boxes in the show column.

  5. Click Map It. You are returned to the Map screen. The vehicle(s) you chose are displayed on the map and are moved to the top of the vehicle grid.
