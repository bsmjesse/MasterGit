OVERVIEW OF THE USER GUIDE

The User Guide helps you use the Sentinel-FM interface on a day-to-day basis. It contains information about working in the following screens:

 

  • Map

  • History

  • Maintenance

  • Messages

  • Reports

 

For information about setting up the Sentinel-FM system, please consult the Configuration Guide.

 

For high-level information about the Sentinel-FM system as well as information about the Help itself, please consult the Getting Started Guide.
