To modify User group access:

 

  • From the button on the left side of any Sentinel-FM page, click Configuration;

  • Select the Users tab and then the User-Groups Assignment sub tab;

  • Select a Group to be modified from the drop-down list;

 

 

  • Move users from the Unassigned users column to the Assigned users column or vice versa by using the Add or Remove buttons;

 

 

Note:  One User can belong to one or more User Groups.  For example, in the case of a User assigned to two User Groups, he/she will have permissions for both.
