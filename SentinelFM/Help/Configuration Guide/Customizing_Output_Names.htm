Modifying Output Names

Outputs trigger actions in a vehicle, for instance, an output can turn the ignition of a car on or off, unlock a door, or honk a horn. For details on outputs, as well as instructions on sending outputs, see Viewing Sensors and Sending Outputs or Commands.

 

Once output sensors are installed in a vehicle, and configured in Sentinel-FM by BSM personnel, their functions do not change, but you can change their names to ones that are more descriptive. You can also change the names for the actions of each output. Common names for outputs might be, for instance, "Door" or "Ignition". Common names for their actions might be "locked" and "unlocked" or "on" and "off".  

 

Note:  A User cannot add or delete outputs; you can only enter and change information for the ones that were installed and configured by BSM personnel.

 

To modify output names for an existing vehicle:

 

  1. From any Sentinel-FM page, click Configuration. Select the Vehicles tab and then the Outputs sub tab and then choose a fleet, then a vehicle. Wait for the list of outputs to appear.

  2. To change the name or settings of an output, click the edit icon () in the row for that output.

  3. Make changes in the following fields:

  4. In the Output field, view the name of the output. If necessary, modify the name to a more descriptive one.

  5. In the Action On field, view the action that will occur if you send this output. If necessary, modify the name to a more descriptive one.

  6. In the Action Off field, view the normal state for this output. If necessary, modify the name to a more descriptive one.
