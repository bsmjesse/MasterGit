Changing Your Password

Under the User Preferences hyperlink on the top right side of the Sentinel-FM Home page (click on Userid or Organization name text in yellow) a User has the option of changing the password that they use to access the Sentinel-FM system.  This is done by selecting the Edit Password button, entering the old password and then entering the new password twice to confirm it and then pressing the Save Password button.

 

Suggestions for choosing a password:

 

  • Do not use an overly simple password, such as "1111". Longer passwords that include both letters and numbers are preferable.

  • Do not use your name.

  • Do not use your user name as a password.

  • Change passwords frequently.
