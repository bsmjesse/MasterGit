Assigning Drivers to Vehicles

Drivers can be assigned to vehicles by an Administrator of the Sentinel FM system.  

 

To assign a driver to a vehicle:

 

  • From the button on the left side of any Sentinel FM page, click Configuration;

  • Select the Drivers tab and then the Vehicle-Drivers Assignment sub tab;

  • Select an Assignment Type, Fleet and Vehicle to be modified from the drop-down lists;

  • Move drivers from the Unassigned Drivers column to the Assigned Drivers column or vice versa by using the Add or Remove buttons;
