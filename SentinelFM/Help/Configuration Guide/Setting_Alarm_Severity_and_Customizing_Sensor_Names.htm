Modifying Input Sensor Names

A variety of sensors (or "inputs") can monitor changes to various parameters in a vehicle.  Physical sensors are wires that are physically connected to different parts of a vehicle's electrical system that sense particular conditions, for instance, whether the ignition of a vehicle is on or off.  Each physical sensor has an "Action On" and an "Action Off" state that can trigger an alarm.   The name of each physical sensor as well as the name and alarm severity of its "Action On" and "Action Off" states can be configured by a User.  

 

There are also several standard virtual sensors that monitor changes to various vehicle parameters through the software in the in-vehicle unit.  These virtual sensors can trigger an alarm if a change in the sensor is detected. For example, an alarm can also be triggered by a vehicle that has a GeoZone assigned to it. For details on GeoZones, see Setting Up GeoZones. The alarm severity of these virtual sensors can be configured by a User.   

Descriptive Names for Physical Sensors

Once physical sensors are installed in a vehicle, their functions do not change, but a User can change the names for the sensors to ones that are more descriptive. A User can also change the names for the On/Off actions of the sensor. Common names for sensors might be, for instance, "Door", "Ignition", or "Panic Button". For a panic button, a User could name the On/Off actions "enabled" and "disabled" or "on" and "off".  

 

Note:  You cannot add or delete sensors; you can only enter and change information for the ones that have been installed and configured by BSM Wireless.

 

 

To modify the physical sensor names for an existing vehicle:

 

  1. From any Sentinel-FM page, click the Configuration hyperlink on the top of the screen. On the Vehicles tab, choose the Sensors/Alarms/Messages sub tab, and choose a fleet, then a vehicle. Wait for the list of sensors to appear.

  2. To change the names of a sensor, click the edit icon () in the row for that sensor.

  3. Make changes in the following fields:

  4. In the Sensor field, view the name of the sensor. If necessary, modify the name to a more descriptive one.

  5. In the Action On - Name field, view the action that may trigger a warning or an alarm for this sensor. If necessary, modify the name to a more descriptive one.

  6. In the Action Off - Name  field, view the normal state for this sensor. If necessary, modify the name to a more descriptive one.
