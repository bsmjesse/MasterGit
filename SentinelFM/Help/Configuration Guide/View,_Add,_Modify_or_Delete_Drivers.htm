View, Add, Modify or Delete Drivers

From the Drivers tab an Administrator of the Sentinel FM system can view, modify, add or delete Users, and assign Users to different access groups.

 

Selecting the Drivers tab and then the Drivers Info sub tab an Administrator can view a table that lists all the Drivers that are associated with a particular User login account.  The table displays the following columns of information for each Driver:

 

ID

First Name

Last Name

License

Assignment

 

To add a new Driver:

 

  • From the hyperlinks on the top of any Sentinel FM page, click Configuration;

  • Select the Drivers tab and then the Drivers Info sub tab;

  • View the table of Drivers that have been entered;

  • Add a new Driver by clicking the Add User button (below the table).

  • Enter information about the driver  in the appropriate fields and click on the Save button to create the new driver.

 

 

Modify an Existing Driver

  • From the hyperlinks on the top of any Sentinel FM page, click Configuration;

  • Select the Drivers tab and then the Drivers Info sub tab;

  • View the table of drivers that have been entered;

  • To modify a driver, in the table, click the edit icon () in the row for that driver.

  • Change desired information for the driver and click on the Save button to save your changes.

 

Delete an Existing Driver

  • From the hyperlinks on the top of any Sentinel FM page, click Configuration;

  • Select the Drivers tab and then the Drivers Info sub tab;

  • View the table of drivers that have been entered;

  • To delete a driver, in the table, click the delete icon () in the row for that driver. You are prompted to confirm the deletion.
