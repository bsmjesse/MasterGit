OVERVIEW OF THE CONFIGURATION GUIDE

The Configuration Guide will help you set up and configure the Sentinel-FM interface. Configure the interface according to the preferences of the person who will be using it the most, generally a Dispatcher.

 

You can:

 

  • Change user preferences

  • Change your password

  • Set up vehicles

  • Modify sensor names and set alarm severity

  • Customize output names

  • Setup up users

  • Assign users to access groups

  • Setup fleets

  • Assign vehicles to fleets

  • Assign users to fleets

  • Setup emails for alarm notification  

  • Set company landmarks

  • Set company GeoZones

  • Assign GeoZones to vehicles

  • Change severity levels for GeoZones

  • View company landmarks and GeoZones

 

For high-level information about the Sentinel-FM interface as well as information about the Help itself, please consult the Getting Started Guide.

 

For information about using the Sentinel-FM interface on a day-to-day basis, please consult the User Guide.
