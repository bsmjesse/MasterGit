View, Add, Modify or Delete Fleets

Selecting the Fleets tab and then the Fleet Management sub tab an Administrator can view a table that lists all the fleets that are associated with a particular User login account.  The table displays the following columns of information for each fleet:

 

Fleet

Description

 

To add a new Fleet:

 

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration;

  • Select the Fleet tab and then the Fleet Management sub tab;

  • View the table of fleets that have been entered;

  • Add a new fleet by clicking the Add Fleet button (below the table).

  • Enter the Fleet Name and Description in the appropriate fields and click on the Save button to create the new fleet.

 

 

Modify an Existing Fleet

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration;

  • Select the Fleet tab and then the Fleet Management sub tab;

  • View the table of fleets that have been entered;

  • To modify a fleet, in the table, click the edit icon () in the row for that fleet.

  • Change desired information for the fleet and click on the Save button to save your changes.

 

Delete an Existing Fleet

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration;

  • Select the Fleet tab and then the Fleet Management sub tab;

  • View the table of fleets that have been entered;

  • To delete a fleet, in the table, click the delete icon () in the row for that fleet. You are prompted to confirm the deletion.
