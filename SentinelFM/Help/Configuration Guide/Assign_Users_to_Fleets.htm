Assign Users to Fleets

In the Sentinel-FM system, you can assign users to different fleets so that specific users only have access to the fleets that are relevant to them.  You can also remove a user's assignment to a specific fleet.  Users can be assigned to more than one fleet at a time.

 

To assign users to a fleet, or remove them from a fleet:

 

  • From any Sentinel-FM page, click on the Configuration hyperlink on the top of the screen.  Select the Fleet tab, and then the Fleet-User Assignment sub tab.

  • From the Fleet drop-down list, choose the fleet you want to work with.  Wait for the lists of assigned and unassigned users to appear.

  • To add a user to the current fleet, highlight the user in the Unassigned users list and click Add->.  To add all users listed to the current fleet, click Add All->.

  • To remove a user from the current fleet, highlight the user in the Assigned users list and click <-Remove.  To remove all users listed from the current fleet, click <-Remove All.

  • The users are moved into the list you choose.
