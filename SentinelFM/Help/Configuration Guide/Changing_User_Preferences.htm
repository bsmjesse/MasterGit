Changing User Preferences

From the User Preferences hyperlink on the top right side of the Sentinel-FM home page (click on Userid or Organization name text in yellow)  a User can set a variety of User-specific parameters including:

 

Setting

Explanation

Time Zone

In order to display local time in the system, you must select your time zone.

 

Km/Miles

Allows User to specify whether distances should be displayed in kilometers or miles.

 

Default Fleet

Allows User to specify default fleet that is automatically pre-selected whenever fleet information is accessed.

 

Alarm Information Refresh Frequency

Allows a User to specify the amount of time before new alarm information on the map screen alarm window is refreshed with new information from the system database.

 

Screen Refresh Frequency

Allows a User to specify the amount of time before the map display and vehicle grid are refreshed with new information from the system database.  

 

Show Landmark Name instead of address

Allows a landmark name instead of an address to be shown on map display, in vehicle grid and in reports.

 

Show Read Messages

Determines whether messages continue to be displayed in the map screen scrolling messages window once they have been read.  By default they are shown there for 24 hours.

 

Automatically adjust for daylight savings time

 

Causes system to automatically adjust for daylight savings time.

Consider position old after

Allows User to specify time after which a vehicle's position is considered old and is displayed as a grey icon

 

 

To change any of these User Preferences the User selects the Edit Preferences button, makes the appropriate changes and then selects the Save Preferences button.
