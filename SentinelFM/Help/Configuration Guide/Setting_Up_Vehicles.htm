View, Add, Modify or Delete Vehicles

This interface allows a User to set up vehicles with descriptions that will be used in the vehicle grid in the Map screen, in History, and in reports.

 

To set up vehicles:

 

  1. From any Sentinel-FM page, click the Configuration hyperlink on the top of the screen.

  2. Select the Vehicles tab and then the Vehicle Info sub tab to view the table of vehicles that have been entered. The table displays the following columns of information for each vehicle.

 

Description

VIN Number

License Plate

Vehicle Type

Make

Model

Year

Color

Box

Email

Icon Type

Add a New Vehicle

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration

  • Select the Vehicles tab and then the Vehicle Info sub tab;

  • View the table of vehicles that have been entered; and,

  • Add a new vehicle by clicking the Add Vehicle button (below the table).

Modify an Existing Vehicle

  • From the hyperlinks on the top of any Sentinel FM page, click Configuration;

  • Select the Vehicles tab and then the Vehicle Info sub tab;

  • View the table of vehicles that have been entered;

  • To modify a vehicle, in the table, click the edit icon () in the row for that vehicle.

  • Change desired information for the vehicle and click on the Save button to save your changes.

 

Delete an Existing Vehicle

  • From the buttons on the left side of any Sentinel FM page, click Configuration;

  • Select the Vehicles tab and then the Vehicle Info sub tab;

  • View the table of vehicles that have been entered;

  • To delete a vehicle, in the table, click the delete icon () in the row for that vehicle. You are prompted to confirm the deletion.
