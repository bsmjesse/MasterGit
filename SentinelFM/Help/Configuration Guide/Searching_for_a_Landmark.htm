Searching for a Landmark

A User can search for a specific landmark from a large database of landmarks by using the search functionality at the top of the landmarks page.

 

 

A User can choose to search by Landmark Name, Description or Address from the drop-down list, then enter some search text and press the Search button.  Only the landmarks meeting the search criteria will be displayed.

 

To clear the search filter a User simply presses the Clear button.
