View, Add, Modify or Delete Users

From the Users tab an Administrator of the Sentinel-FM system can view, modify, add or delete Users, and assign Users to different access groups.

 

Selecting the Users tab and then the User Info sub tab an Administrator can view a table that lists all the Users that are associated with a particular User login account.  The table displays the following columns of information for each user:

 

First Name

Last Name

User Name

Expired Date

 

To add a new User:

 

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration;

  • Select the Users tab and then the User Info sub tab;

  • View the table of Users that have been entered;

  • Add a new User by clicking the Add User button (below the table).

  • Enter the First Name, Last Name, User Name, Expiration Date, and password in the appropriate fields and click on the Save button to create the new user.

 

 

Modify an Existing User

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration;

  • Select the Users tab and then the User Info sub tab;

  • View the table of users that have been entered;

  • To modify a user, in the table, click the edit icon () in the row for that user.

  • Change desired information for the user and click on the Save button to save your changes.

 

Delete an Existing User

  • From the hyperlinks on the top of any Sentinel-FM page, click Configuration;

  • Select the Users tab and then the User Info sub tab;

  • View the table of users that have been entered;

  • To delete a user, in the table, click the delete icon () in the row for that user. You are prompted to confirm the deletion.
