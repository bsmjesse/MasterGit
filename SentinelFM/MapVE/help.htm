BSM Bing Maps v1.1.1

Navigation

  • Pressing the `Ctrl` key and click dragging the mouse will create a `rubber-band` box that will zoom to bounds after mouse button is released
  • Real-time Traffic flow visibility can be toggled by clicking the `Traffic` button in the command bar
    (Traffic flow is subject to availability in certain metropolitan areas)

    • Fleets

  • Fleet layer visibility can be toggled by clicking the `Fleet` button in the command bar
  • Zoom to Unit can be activated by clicking on unit icon
  • Unit information can be made visible while hovering over unit icons

    • Landmarks

  • Landmark data for the current area can be accessed by clicking on `Landmarks` button in the command bar
  • `Landmark` button is only available at specific zoom levels
  • Zoom to Landmark can be activated by clicking on Landmark icon
  • Landmark information can be made visible while hovering over Landmark icons
  • Landmark radius with Zoom can be made visible by right-clicking Landmark icons
  • Landmark radius without Zoom can be made visible by right-clicking Landmark icons while holding the `Shift` key down

    • Geozones

  • Geozone data for the current area can be accessed by clicking on `Geozones` button in the command bar
  • `Geozones` button is only available at specific zoom levels
  • Zoom to Geozone can be activated by clicking on Geozone icon
  • Geozone information can be made visible while hovering over Geozone icons
  • Geozone polygon can be made visible by right-clicking Geozone icons
